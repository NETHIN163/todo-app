# 📝 To-Do App

## 🚀 Description
An advanced and interactive To-Do App built using HTML, CSS, and JavaScript.  
This application allows users to manage daily tasks efficiently with a clean and modern UI.

---

## ✨ Features
- ➕ Add new tasks  
- ✅ Mark tasks as completed  
- ❌ Delete tasks  
- ✏️ Edit tasks  
- 🔍 Search tasks  
- 🌙 Dark/Light mode toggle  
- 📊 Task counter (Total & Completed)  
- 💾 Persistent storage using localStorage  

---

## 🎨 UI Highlights
- Modern Glassmorphism design  
- Responsive layout  
- Smooth and clean user experience  

---

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  
- localStorage
